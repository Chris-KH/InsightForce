# Insight Forge - Frontend Web Dashboard

[cite_start]This repository contains the frontend codebase for **Insight Forge**, an Agentic AI Ecosystem designed to actively collaborate with content creators[cite: 84, 87]. [cite_start]The frontend serves as a Web SaaS Application (Web Dashboard) that acts as the primary communication channel between the user and the multi-agent backend system[cite: 444, 592].

## 🚀 Tech Stack

The frontend is built upon a high-performance, component-based architecture using the following technologies:

- [cite_start]**React & Vite:** Provides a fast development workflow and highly responsive user interface[cite: 404].
- [cite_start]**TypeScript:** Employed to provide strict type safety, enhancing code maintainability and reducing potential runtime errors[cite: 406].
- [cite_start]**shadcn/ui:** A UI component library built on Radix UI and Tailwind CSS, utilized to create accessible, reusable, and consistent interface elements across the application[cite: 405].

## ✨ Key Features

[cite_start]The UI Frontend App is designed to facilitate a Human-in-the-Loop (HITL) control architecture, ensuring creators retain ultimate authority over their channels[cite: 346, 442].

- [cite_start]**Channel Overview Dashboard:** Displays high-level metrics including Total Views, Revenue, and Average Sentiment[cite: 459, 464, 468, 470].
- [cite_start]**Human-in-the-Loop Interface:** Collects user prompts and allows creators to explicitly approve, revise, or reject actions proposed by the AI agents before execution[cite: 446, 448].
- [cite_start]**AI Suggestions Panel:** Presents real-time content strategy, thumbnail optimization, and audience engagement recommendations generated by the Content Architect Agent[cite: 341, 485, 486, 490].
- [cite_start]**Channel Health Monitoring:** Visualizes recent top-performing videos alongside their respective views, reactions, and sentiment[cite: 467, 472].
